A very simple image carrousel plugin which allows you to create slides with images and links and choose which ones to display on your posts and pages fro ma meta box.

== Description ==

The plugin does not come with multiple transsitions and settings. All you have to do is create and choose your slides.
Create a global collections of slides and choose which ones to display on single posts by selecting them and adding this short code: [rsris_slider]

New layout with live search in the list of slides and with thumbnails.

You can now embed YouTube videos in your slides.

The slider is responsive and will addapt to your theme.

You can manually add slides via the shortcode  attribute 'slides' like so [rsris_slider slides="ID's OF SLIDES].

For more informaiton see [this page.](http://bakadesign.dk/really-simple-responsive-images-slider-for-wordpress/ "rsris on bakadesign")

Warning: Internet Explorer 8 and earlier not fully supported.


== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place `[rsris_slider]` in your editor for displaying the carrousel
